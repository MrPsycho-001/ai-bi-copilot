# AI Business Intelligence Copilot

An enterprise-grade, Agentic AI platform that automates data engineering, business analysis, and strategic forecasting. 

## Architecture
- **Frontend:** Next.js (App Router), React, TailwindCSS
- **Backend:** FastAPI, Python, PostgreSQL
- **AI Orchestration:** LangGraph, Gemini Pro API
- **Data Engine:** Pandas, NumPy, Prophet

## Local Development
1. Start the database: docker compose up -d
2. Configure environment variables (see .env.example).
3. Start backend and frontend services (instructions to follow in development phases).
